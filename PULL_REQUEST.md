#  Pull Request Template

Thank you for contributing to **Ressurected-Bird**! This project grows stronger because of community support. Please use this template to ensure your pull request is clear, inclusive, and easy for everyone to review.

---

## 1. Summary

Provide a friendly, easy-to-understand summary of what this pull request does.

* What problem does it solve?
* How does it help the community or improve the project?

---

## 2. Description of Changes

Explain the changes you made:

* What was added, updated, or removed?
* Why is this change useful for users or contributors?
* Highlight any improvements to accessibility, clarity, or usability.

---

## 3. Motivation

Tell us **why** you created this PR:

* Was it inspired by an Issue?
* Was it suggested by someone in the community?
* Personal motivation or project improvement?

---

## 4. Related Issues / Discussions

Please reference any:

* Reported issues (Closes # )
* Discussions
* Community suggestions

This helps future contributors understand why this PR exists.

---

Check all that apply:

* [ ] New Feature

* [ ] UI/UX Enhancement
* [ ] Accessibility Improvement
* [ ] Community Suggestion Implementation

## 6. Community-Focused Checklist

### Quality & Inclusivity

* [ ] The contribution is beginner-friendly and easy to understand
* [ ] Naming is clear and human-readable
* [ ] No unnecessary complexity added
* [ ] Improves user or contributor experience

### Documentation & Communication

* [ ] All changes are clearly documented
* [ ] Comments added where helpful
* [ ] Screenshots included for UI/UX changes

### Accessibility

* [ ] Colour contrast and readability improved (if applicable)
* [ ] Keyboard-navigation friendly (if applicable)
* [ ] Images include alt-text
